# Saucy Invoice Maker

A web-based application for generating, previewing, and downloading professional invoices. It provides a real-time editor alongside a live preview of the document.

## Features

- **Real-time Editing**: Modify sender details, client information, and invoice terms with immediate visual feedback.
- **Line Item Management**: Add, edit, and remove items. Subtotals and grand totals are calculated automatically.
- **PDF Export**: Generate and download the invoice as a PDF file directly from the browser.
- **Print Optimized**: dedicated styles ensure the invoice prints correctly on standard A4 paper.
- **Responsive Interface**: Features a split-view layout on desktop and a stacked layout on mobile devices.

## Setup

1. Clone the repository.
2. Install the dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.
